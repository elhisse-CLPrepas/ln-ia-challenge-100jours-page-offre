# Challenge Formation 100 jours — Atelier LN-IA

Page publique de présentation du Challenge Formation 100 jours.

## Objectif

Présenter l'offre, les avantages, les 5 phases, les livrables et l'appel à rejoindre le challenge.

## Test local

```powershell
python -m http.server 8080
```

Puis ouvrir :

```text
http://localhost:8080
```

## GitHub Pages

Dans GitHub :

1. Aller dans `Settings`.
2. Ouvrir `Pages`.
3. Choisir `Deploy from a branch`.
4. Sélectionner la branche `main`.
5. Choisir `/ (root)`.
